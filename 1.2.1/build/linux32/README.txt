Steps for building TerraME under LINUX

1 - Download and compile dependencies. (Read instructions from dependencies folder)
2 - cd to build/linux32
3 - run ../../dependencies/qt/bin/qmake TerraME.pro
4 - run make
5 - run ./binaries
6 - Use InstallJammer and select the 'linux32' installer project from 'install' folder
